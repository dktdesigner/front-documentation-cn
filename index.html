# 首页（Homepage)

# 滚屏（Banner）

# DSL

  • 填写 排期表 ,请注意填写以下设计师作图所用到的素材（标题，副标题，主推产品信息等），以及活动页面的链接。
  • 活动页面如果是落地页（此类页面开发和设计都需要1周时间）
  • 请确保上线前2周，所有信息填写完毕。

# Designer

  • 根据 排期表 及 模板 作图，PC，Mobile 有2张图。
  • 图片导出为web所用格式的JPG图片 ，质量80。
  • 命名规则： SportsName_n_MB.jpg (运动名称，n表示第几屏，PC/MB)
  • 在前一周周五之前，请将图片上传到文件夹。

# Developer

  • 上传图片到PIXL。
  • 使用Tracking工具生成追踪码。
  • 每周一根据 排期表 及 文件夹 上传到官网

# 侬说搭配楼层 (Panoply Floor)

# DSL

  • 填写 排期表 ,请注意填写以下设计师作图所用到的素材（标题，副标题，4/6个产品信息等），以及listing链接。
  • 第一第二个楼层PC图片的链接是listing，MB图片的链接为第一个产品的链接。第三个楼层PC和MB链接相同。第一个楼层填写4个产品，以及一条运动TIP，请填写标题和正文内容。第二第三个楼层可以填写6个产品。
  • 请确保上线前2周，所有信息填写完毕。

# Designer

  • 根据 排期表 及 模板 作图，PC，Mobile 有2张图。其中第三个楼层的MB图和第一第二个图不同
  • 图片导出为web所用格式的JPG图片 ，质量80。
  • 命名规则： SportsName_n_MB.jpg (运动名称，n表示panoply的第几个楼层[一共3层]，PC/MB)
  • 在前一周周五之前，请将图片上传到文件夹。

# Developer

  • 上传图片到PIXL。
  • 每周一根据 排期表 及 文件夹 上传到官网
